前言

安装前言

本指南旨在为初次安装 Arch Linux 的用户提供一份清晰、循序渐进的安装教程。Arch Linux 是一个轻量级、高度可定制的 Linux 发行版，遵循“KISS”（保持简单与直接）原则。它采用滚动更新模式，为用户提供了最新的软件包和一个由用户主导的、高度自由的系统构建体验。

常见问题：为何选择 Arch Linux？

Q：相比 Windows 和其它 Linux 发行版有什么优势或特点？

A：Arch Linux 的设计哲学带来了一些独特之处：

  • 极简与掌控：安装介质仅提供一个最小化的基础系统，用户可以根据自身需求，从零开始安装和配置每一个组件，从而获得一个完全符合个人预期的系统，避免不必要的预装软件。

  • 滚动更新：通过持续的软件包更新，可以快速获得最新的内核、驱动和应用程序，无需等待大版本升级。

  • 强大的软件生态：官方的软件仓库 (pacman) 配合用户仓库 (AUR)，提供了极其丰富的软件选择。

  • 卓越的文档：Arch Wiki 被广泛认为是 Linux 领域最全面、最优质的社区文档之一。

警告：若您初次安装并使用 Arch Linux，请按照此文档谨慎操作。

安装过程需要您直接与命令行交互，并对磁盘分区、网络配置等系统基础知识有一定了解。请务必仔细阅读每一步的说明，理解其含义后再执行操作。错误的命令可能导致数据丢失或系统无法启动。

本 Wiki 页面将引导您完成从准备安装介质、分区、安装基本系统、配置引导程序到进入新系统的全过程。安装基本系统后，您可以根据后续指南配置图形界面（桌面环境）与常用软件。请确保在开始前已备份所有重要数据，并准备好稳定的网络连接。

Preface

This guide aims to provide first-time Arch Linux users with a clear, step-by-step installation tutorial. Arch Linux is a lightweight, highly customizable Linux distribution that adheres to the “KISS” (Keep It Simple, Stupid) principle. It employs a rolling release model, offering users the latest software packages and a user-driven, highly flexible system-building experience.

FAQ: Why Choose Arch Linux?

Q: What are the advantages or characteristics compared to Windows and other Linux distributions?

A: Arch Linux’s design philosophy brings several distinctive features:

  • Minimalism and Control: The installation media provides only a minimal base system. Users can install and configure every component from scratch according to their own needs, resulting in a system that perfectly matches personal expectations, avoiding unnecessary pre-installed software.

  • Rolling Release: Through continuous package updates, users can quickly obtain the latest kernel, drivers, and applications without waiting for major version upgrades.

  • Powerful Software Ecosystem: The official software repository (pacman) combined with the Arch User Repository (AUR) offers an extremely rich selection of software.

  • Excellent Documentation: The Arch Wiki is widely regarded as one of the most comprehensive and high-quality community documentation resources in the Linux world.

Warning: If you are installing and using Arch Linux for the first time, please follow this document carefully.

The installation process requires direct interaction with the command line and a basic understanding of system fundamentals such as disk partitioning and network configuration. Be sure to read the instructions for each step carefully and understand their meaning before executing any commands. Incorrect commands may lead to data loss or a system that fails to boot.

This Wiki page will guide you through the entire process, from preparing the installation media, partitioning, installing the base system, configuring the bootloader, to entering the new system. After installing the base system, you can follow subsequent guides to configure the graphical interface (desktop environment) and common software. Please ensure that you have backed up all important data before starting and have a stable network connection ready.
