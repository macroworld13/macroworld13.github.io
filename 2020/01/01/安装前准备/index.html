安装前准备

1. 下载 Arch Linux 镜像

Arch Linux 官方下载地址：https://archlinux.org/download/

当前最新稳定版本为 2026.03.01，包含内核版本 6.18.13，ISO 镜像大小 1.4 GB。推荐通过 BitTorrent 方式下载（速度更快且能保障文件完整性），具体操作如下：

在下载页面找到 BitTorrent Download (recommended) 板块：

  • 任选其中一个选项（Magnet 链接或 Torrent 文件），使用支持 DHT 的 BT 客户端（如 qBittorrent、Transmission）下载；
  • 下载完成后建议验证文件完整性：可通过官方提供的 sha256sums.txt 或 b2sums.txt 校验，也可通过 PGP 签名验证（具体命令参考官方文档）。

若无法使用 BT 下载，可选择 HTTP 直接下载，国内推荐镜像源：

  • 清华大学：https://mirrors.tuna.tsinghua.edu.cn/archlinux/iso/
  • 中国科学技术大学：https://mirrors.ustc.edu.cn/archlinux/iso/
  • 南京大学：https://mirrors.nju.edu.cn/archlinux/iso/

2. 制作安装介质（Live 环境）

方式一：使用 Ventoy 制作启动盘（推荐）

Ventoy 是一款免格式化的启动盘制作工具，支持直接拷贝 ISO 镜像即可启动，操作简单且兼容多系统镜像。

  1. 下载 Ventoy 安装包：https://www.ventoy.net/cn/download.html

    • 最新版本：1.1.10（2025-12-21 发布）
    • Windows 系统下载 ventoy-1.1.10-windows.zip（15 MB）；
    • Linux 系统下载 ventoy-1.1.10-linux.tar.gz（19 MB）；
    • 国内下载镜像（推荐，速度更快）：
      • 南京大学镜像站：https://mirrors.nju.edu.cn/github-release/ventoy/Ventoy
      • 百度网盘：https://pan.baidu.com/s/1lVoctl45GpLBgG53W1W7qQ?pwd=vtoy（提取码：vtoy）

  2. 制作步骤：

    • ⚠️ 重要提醒：提前备份 U 盘内所有重要数据（制作过程会格式化 U 盘）；
    • 解压 Ventoy 安装包，运行对应系统的安装程序（Windows 运行 Ventoy2Disk.exe，Linux 执行 sudo sh Ventoy2Disk.sh）；
    • 选择要制作的 U 盘，点击“安装”，等待完成（无需手动分区格式化）；
    • 将下载好的 Arch Linux ISO 镜像文件直接拷贝到 U 盘根目录（无需解压），即可完成启动盘制作。

方式二：压缩卷（没有 U 盘的情况下）

适用于无 U 盘时，通过硬盘分区制作启动环境：

  1. 打开 Windows 磁盘管理（右键此电脑 → 管理 → 磁盘管理）；
  2. 选择一个剩余空间充足的分区（建议剩余空间 ≥ 2 GB），右键点击“压缩卷”，输入压缩空间大小（需大于 Arch Linux ISO 镜像大小，建议 2 GB 以上）；
  3. 右键新生成的“未分配”空间，选择“新建简单卷”，按照向导完成创建（文件系统需选择 FAT32，分配盘符可自定义）；
  4. 用解压工具（如 7-Zip、WinRAR）打开 Arch Linux ISO 镜像，将镜像内所有文件复制到刚刚新建的 FAT32 分区内，完成启动环境部署。

3. 关闭安全启动

Arch Linux 安装过程中需要关闭主板的安全启动（Secure Boot），否则可能无法正常引导：

  1. 重启电脑，开机时持续按下对应的 BIOS 进入键（不同主板按键不同，常见为 ESC、F2、DELETE、F10，开机时屏幕通常会提示）；
  2. 进入 BIOS 界面后，找到“Security”（安全）选项卡，找到“Secure Boot”（安全启动）选项，设置为“Disabled”（关闭）；
  3. 保存设置并退出 BIOS（通常按 F10 保存，部分主板需选择“Save and Exit”）。

4. 选择启动项

  1. 保存 BIOS 设置后电脑会重启，再次开机时持续按下启动项选择键（常见为 F12、F8、ESC，具体按键可参考主板说明书）；
  2. 在启动项列表中，选择制作好的安装介质（Ventoy 制作的 U 盘通常显示为“UEFI: [U盘名称]”，硬盘分区制作的启动环境显示为对应的 FAT32 分区盘符）；
  3. 选择后回车，系统会从安装介质载入 Live 环境（类似 Windows PE，无需安装即可运行），等待加载完成后进入命令行界面，即准备好安装环境。

⚠️ 注意：本教程仅涉及 UEFI 固件下的安装，传统 BIOS（Legacy）固件下的分区表、引导配置等方法有所不同，若你的电脑为 BIOS 固件，请参考 Arch Wiki 对应教程。
